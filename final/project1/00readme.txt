Để có thể compile các file TestCaseX.cpp
Các file TestCaseX.cpp này phải cùng cấp với các HeaderFile (.hpp, .cpp) - thư viện tự định nghĩa

-- Chạy terminal trong folder chứa file TestCaseX.cpp
-- Chạy lệnh biên dịch: 
                        g++ TestCaseX.cpp -o TestCaseX.exe
-- Có thể chạy file .exe bằng 2 cách sau: 
        C1: chạy chương trình .exe bằng click đúp vào file .exe sau khi đã tạo
        C2: chạy terminal: 
                         & .\'TestCaseX.exe'
